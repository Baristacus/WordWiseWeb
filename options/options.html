Word Wise Web: 도움말

1. 기본 사용법
  1. 단어 선택: 웹 페이지에서 모르는 단어를 더블 클릭하거나 드래그하여 선택합니다.
  2. 단어 저장: 선택한 단어 옆에 나타나는 플로팅 아이콘을 클릭하면 단어가 개인 단어장에 저장되며, 의미를 확인할 수 있습니다.
2. 주요 기능
2.1 팝업 창
  • 단어 추가: 직접 단어를 입력하여 의미를 검색하고 저장할 수 있습니다.
  • 최근 단어: 최근에 저장한 단어 목록을 확인할 수 있습니다.
  • 단어 관리: 저장된 단어를 삭제하거나 수정할 수 있습니다.
2.2 옵션 페이지
  • 단어장: 저장한 모든 단어를 관리하고 학습할 수 있습니다.
  • 학습하기: 저장한 단어로 퀴즈를 풀고 학습 진도를 확인할 수 있습니다.
  • 설정: 확장 프로그램의 동작을 사용자 취향에 맞게 조정할 수 있습니다.
3. 고급 설정
  • API 키 설정: Google Gemini API 키를 설정하여 정확한 단어 의미를 제공받을 수 있습니다.
  • 플로팅 아이콘: 단어 선택 시 나타나는 아이콘의 표시 여부를 설정할 수 있습니다.
  • 예문 저장: 단어와 함께 예문을 저장할지 설정할 수 있습니다.
  • 단어 하이라이트: 이미 저장한 단어를 웹 페이지에서 하이라이트로 표시할 수 있습니다.
4. 문제 해결
단어가 저장되지 않아요
  • 인터넷 연결을 확인해 주세요.
  • API 키가 올바르게 설정되어 있는지 확인해 주세요.
플로팅 아이콘이 나타나지 않아요
  • 설정에서 플로팅 아이콘 표시 옵션이 켜져 있는지 확인해 주세요.
  • 페이지를 새로고침 해보세요.
의미가 정확하지 않아요
  • 선택한 단어의 맥락을 고려하여 의미를 제공하고 있습니다. 다른 맥락에서 단어를 선택해 보세요.
  • API 키를 확인하고, 필요하다면 재설정해 보세요.
5. 피드백 및 지원

Word Wise Web을 개선하는데 도움을 주세요!

  • 버그 리포트: 문제를 발견하셨다면 자세한 내용과 함께 알려주세요.
  • 기능 제안: 새로운 아이디어가 있으시다면 언제든 공유해 주세요.
  • 문의하기: 추가 도움이 필요하시면 언제든 문의해 주세요.
